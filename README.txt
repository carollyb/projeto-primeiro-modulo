
Projeto - Primeiro Módulo [JOGO DE ESCOLHAS]

Finalização do primeiro módulo do curso de capacitação em Web Development Fullstack, com jogo de escolhas utilizando estruturas condicionais, entradas de usuário e funções e eventos em JavaScript, além de outros conhecimentos adquiridos ao longo do módulo: HTML (marcação de texto, botão) e CSS (flexbox, manipulação de cores e imagens).
